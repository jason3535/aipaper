摘要 · Abstract

大型语言模型（LLM）彻底改变了人工智能领域，使得以前被认为只有人类才能完成的自然语言处理任务成为可能。在这项工作中，我们介绍了Qwen，这是我们大型语言模型系列的第一部分。Qwen是一个全面的语言模型系列，包含不同参数量的不同模型。它包括Qwen（基础预训练语言模型）和Qwen-Chat（使用人类对齐技术微调的聊天模型）。基础语言模型在众多下游任务中持续展现出优越的性能，而聊天模型，特别是那些使用基于人类反馈的强化学习（RLHF）训练的模型，具有很强的竞争力。聊天模型具备先进的工具使用和规划能力，可用于创建智能体应用，即使在复杂任务（如使用代码解释器）上与更大的模型相比也表现出色。此外，我们还开发了专门用于编码的模型Code-Qwen和Code-Qwen-Chat，以及专注于数学的模型Math-Qwen-Chat，它们都基于基础语言模型构建。这些模型与开源模型相比表现出显著改进的性能，略逊于专有模型。

Large language models (LLMs) have revolutionized the field of artificial intelligence, enabling natural language processing tasks that were previously thought to be exclusive to humans. In this work, we introduce Qwen, the first installment of our large language model series. Qwen is a comprehensive language model series that encompasses distinct models with varying parameter counts. It includes Qwen, the base pretrained language models, and Qwen-Chat, the chat models finetuned with human alignment techniques. The base language models consistently demonstrate superior performance across a multitude of downstream tasks, and the chat models, particularly those trained using Reinforcement Learning from Human Feedback (RLHF), are highly competitive. The chat models possess advanced tool-use and planning capabilities for creating agent applications, showcasing impressive performance even when compared to bigger models on complex tasks like utilizing a code interpreter. Furthermore, we have developed coding-specialized models, Code-Qwen and Code-Qwen-Chat, as well as mathematics-focused models, Math-Qwen-Chat, which are built upon base language models. These models demonstrate significantly improved performance in comparison with open-source models, and slightly fall behind the proprietary models.

核心贡献 · Key contributions

  • 提出Qwen，一个包含不同参数规模的全面大语言模型系列。
    Introduces Qwen, a comprehensive LLM series with varying parameter counts.
  • 基础预训练模型在多个下游任务中表现优异。
    Base pretrained models show superior performance across multiple downstream tasks.
  • 使用RLHF训练的聊天模型达到高度竞争性表现。
    Chat models trained with RLHF achieve highly competitive performance.
  • 聊天模型具备高级工具使用和规划能力，适用于智能体应用。
    Chat models possess advanced tool-use and planning for agent applications.
  • 开发了Code-Qwen和Math-Qwen-Chat等专用模型。
    Develops Code-Qwen and Math-Qwen-Chat specialized models.
  • 专用模型显著优于开源模型，接近专有模型性能。
    Specialized models significantly outperform open-source models and approach proprietary ones.

局限 · Limitations

  • 在复杂任务上性能略逊于专有模型。
    Performance on complex tasks slightly falls behind proprietary models.
  • 在超大规模或实际部署场景下的评估有限。
    Limited evaluation on extremely large-scale or real-world deployment scenarios.
  • 潜在的偏见和安全问题未完全解决。
    Potential biases and safety issues not fully addressed.
  • 对非英语语言的泛化能力可能有限。
    Generalization to non-English languages may be limited.
