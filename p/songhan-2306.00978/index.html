AWQ：面向大语言模型压缩与加速的激活感知权重量化

AWQ: Activation-aware Weight Quantization for LLM Compression and Acceleration

韩松 Song Han · MIT · 2023-06-01 · arXiv:2306.00978 ↗ · 被引 1478

摘要 · Abstract

大语言模型（LLM）已变革众多AI应用。设备端LLM日益重要：在边缘设备本地运行LLM可降低云计算成本并保护用户隐私。然而，巨大的模型规模和有限的硬件资源带来了显著的部署挑战。我们提出激活感知权重量化（AWQ），一种面向LLM低位仅权重量化的硬件友好方法。AWQ发现并非LLM中所有权重同等重要。仅保护1%的显著权重即可大幅降低量化误差。为识别显著权重通道，应参考激活分布而非权重。为避免硬件低效的混合精度量化，我们通过数学推导得出缩放显著通道可降低量化误差。AWQ采用等价变换缩放显著权重通道以保护它们。缩放因子通过离线收集激活统计信息确定。AWQ不依赖任何反向传播或重构，因此能泛化至不同领域和模态而不过拟合校准集。AWQ在各种语言建模和领域特定基准（编码与数学）上优于现有工作。得益于更好的泛化性，它在指令调优语言模型上实现了出色的量化性能，并首次在多模态语言模型上取得优异效果。伴随AWQ，我们实现了TinyChat，一个专为4位设备端LLM/VLM设计的高效灵活推理框架。通过内核融合和平台感知权重打包，TinyChat在桌面和移动GPU上相比Huggingface FP16实现实现了超过3倍的加速。它还使得70B Llama-2模型在移动GPU上的部署变得普及。

Large language models (LLMs) have transformed numerous AI applications. On-device LLM is becoming increasingly important: running LLMs locally on edge devices can reduce the cloud computing cost and protect users' privacy. However, the astronomical model size and the limited hardware resource pose significant deployment challenges. We propose Activation-aware Weight Quantization (AWQ), a hardware-friendly approach for LLM low-bit weight-only quantization. AWQ finds that not all weights in an LLM are equally important. Protecting only 1% salient weights can greatly reduce quantization error. To identify salient weight channels, we should refer to the activation distribution, not weights. To avoid the hardware-inefficient mix-precision quantization, we mathematically derive that scaling up the salient channels can reduce the quantization error. AWQ employs an equivalent transformation to scale the salient weight channels to protect them. The scale is determined by collecting the activation statistics offline. AWQ does not rely on any backpropagation or reconstruction, so it generalizes to different domains and modalities without overfitting the calibration set. AWQ outperforms existing work on various language modeling and domain-specific benchmarks (coding and math). Thanks to better generalization, it achieves excellent quantization performance for instruction-tuned LMs and, for the first time, multi-modal LMs. Alongside AWQ, we implement TinyChat, an efficient and flexible inference framework tailored for 4-bit on-device LLM/VLMs. With kernel fusion and platform-aware weight packing, TinyChat offers more than 3x speedup over the Huggingface FP16 implementation on both desktop and mobile GPUs. It also democratizes the deployment of the 70B Llama-2 model on mobile GPUs.

核心贡献 · Key contributions

  • 提出激活感知权重量化（AWQ），用于大语言模型的低位权重量化。
    Proposes Activation-aware Weight Quantization (AWQ) for LLM low-bit weight-only quantization.
  • 发现仅基于激活分布保护1%的显著权重即可大幅降低量化误差。
    Identifies that protecting only 1% salient weights based on activation distribution reduces quantization error.
  • 推导出逐通道缩放方法，在不引入混合精度硬件开销的前提下保护显著权重。
    Derives a per-channel scaling method to protect salient weights without mixed-precision hardware overhead.
  • 在语言建模、指令微调及多模态语言模型上取得领先的量化性能。
    Achieves state-of-the-art quantization performance on language modeling, instruction-tuned, and multi-modal LMs.
  • 实现TinyChat推理框架，在桌面和移动GPU上相比FP16实现3倍以上加速。
    Implements TinyChat inference framework with 3x+ speedup over FP16 on desktop and mobile GPUs.
  • 首次实现70B Llama-2模型在移动GPU（Jetson Orin）上的部署。
    Enables deployment of 70B Llama-2 on mobile GPU (Jetson Orin) for the first time.

局限 · Limitations

  • AWQ专注于权重量化，未涉及激活量化。
    AWQ focuses on weight-only quantization; activation quantization is not addressed.
  • 该方法需要从预训练数据中选取小规模校准集以避免过拟合。
    The method requires a small calibration set from pre-training data to avoid overfitting.
  • INT2量化性能与FP16基线相比仍存在差距。
    Performance at INT2 quantization still shows a gap compared to FP16 baseline.
  • 缩放搜索空间限于单一超参数alpha，可能并非所有层的最优解。
    The scaling search space is limited to a single hyperparameter alpha, which may not be optimal for all layers.
  • 评估主要基于LLaMA和OPT系列，其他架构可能需要进一步验证。
    Evaluation is primarily on LLaMA and OPT families; other architectures may require further validation.
