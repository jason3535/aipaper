NVIDIA Nemotron Nano 2: An Accurate and Efficient Hybrid Mamba-Transformer Reasoning Model

摘要 · Abstract

我们推出了Nemotron-Nano-9B-v2，这是一种混合Mamba-Transformer语言模型，旨在提高推理工作负载的吞吐量，同时在与类似规模模型相比时实现最先进的准确率。Nemotron-Nano-9B-v2基于Nemotron-H架构，其中常见Transformer架构中的大部分自注意力层被替换为Mamba-2层，从而在生成推理所需的长思考轨迹时实现更快的推理速度。我们首先使用FP8训练配方在20万亿个token上预训练了一个120亿参数的模型（Nemotron-Nano-12B-v2-Base）。在对Nemotron-Nano-12B-v2-Base进行对齐后，我们采用Minitron策略对模型进行压缩和蒸馏，目标是在单个NVIDIA A10G GPU（22GiB内存，bfloat16精度）上支持多达128k个token的推理。与现有类似规模的模型（例如Qwen3-8B）相比，我们展示了Nemotron-Nano-9B-v2在推理基准测试中达到了同等或更好的准确率，同时在推理设置（如8k输入和16k输出token）下实现了高达6倍的推理吞吐量。我们将在Hugging Face上发布Nemotron-Nano-9B-v2、Nemotron-Nano-12B-v2-Base和Nemotron-Nano-9B-v2-Base检查点，以及我们大部分预训练和后训练数据集。

We introduce Nemotron-Nano-9B-v2, a hybrid Mamba-Transformer language model designed to increase throughput for reasoning workloads while achieving state-of-the-art accuracy compared to similarly-sized models. Nemotron-Nano-9B-v2 builds on the Nemotron-H architecture, in which the majority of the self-attention layers in the common Transformer architecture are replaced with Mamba-2 layers, to achieve improved inference speed when generating the long thinking traces needed for reasoning. We create Nemotron-Nano-9B-v2 by first pre-training a 12-billion-parameter model (Nemotron-Nano-12B-v2-Base) on 20 trillion tokens using an FP8 training recipe. After aligning Nemotron-Nano-12B-v2-Base, we employ the Minitron strategy to compress and distill the model with the goal of enabling inference on up to 128k tokens on a single NVIDIA A10G GPU (22GiB of memory, bfloat16 precision). Compared to existing similarly-sized models (e.g., Qwen3-8B), we show that Nemotron-Nano-9B-v2 achieves on-par or better accuracy on reasoning benchmarks while achieving up to 6x higher inference throughput in reasoning settings like 8k input and 16k output tokens. We are releasing Nemotron-Nano-9B-v2, Nemotron-Nano12B-v2-Base, and Nemotron-Nano-9B-v2-Base checkpoints along with the majority of our pre- and post-training datasets on Hugging Face.

核心贡献 · Key contributions

  • 提出 Nemotron-Nano-9B-v2，一种用于高效推理的混合 Mamba-Transformer 模型。
    Introduces Nemotron-Nano-9B-v2, a hybrid Mamba-Transformer model for efficient reasoning.
  • 用 Mamba-2 层替换大部分自注意力层，以提升长思维链的推理速度。
    Replaces most self-attention layers with Mamba-2 layers to improve inference speed for long thinking traces.
  • 使用 FP8 训练方案在 20 万亿 token 上预训练 12B 参数模型。
    Pre-trains a 12B-parameter model on 20 trillion tokens using FP8 training recipe.
  • 应用 Minitron 压缩与蒸馏，使单张 A10G GPU 上支持 128k token 推理。
    Applies Minitron compression and distillation to enable 128k token inference on a single A10G GPU.
  • 在推理基准上达到与 Qwen3-8B 相当或更优的准确率。
    Achieves on-par or better accuracy on reasoning benchmarks compared to Qwen3-8B.
  • 在 8k 输入和 16k 输出 token 的推理场景中，实现高达 6 倍的推理吞吐量提升。
    Demonstrates up to 6x higher inference throughput for reasoning settings with 8k input and 16k output tokens.

局限 · Limitations

  • 评估仅限于推理基准，其他任务未评估。
    Evaluation limited to reasoning benchmarks; other tasks not assessed.
  • 吞吐量仅与 Qwen3-8B 对比，需更广泛比较。
    Throughput comparison only with Qwen3-8B; broader comparisons needed.
  • 模型大小和内存限制可能影响在较小 GPU 上的部署。
    Model size and memory constraints may limit deployment on smaller GPUs.
  • 训练数据和方案细节未完全公开，影响可复现性。
    Training data and recipe details not fully disclosed for reproducibility.
  • 超过 128k token 的长上下文性能未验证。
    Long-context performance beyond 128k tokens not verified.
